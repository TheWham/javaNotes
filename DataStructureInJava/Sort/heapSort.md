## 																																																																									堆排序

>   堆排序的思路 :
>
> 堆排序主要是按照二叉树的形式,先将二叉树变成大堆 (也就是树的 非叶子节点 >= 它的左右子节点)小堆反之.
>
> 大堆的实现方法是先找到树的非子叶节点,然后再比较它的左右子节点, 把较大的子节点与 它的非子叶节点(即父节点)互换
>
> 进行以上操作直到二叉树变成大堆.
>
> 接着再把树顶节点也就是最大的节点与树尾节点交换,然后在进行以上操作把二叉树变成大堆,然后再交换首尾,直到二叉树变成有序的. 

代码实现准备 : 

需要找到非子叶节点,二叉树的最后一个非子叶节点是 ``arr.length / 2  - 1`` 然后它的左右子节点分别是``i * 2  + 1`` 和 ``i * 2 + 2``  然后 i 是0 开始. 需要注意的是 刚开始将二叉树变成大堆时是从最后一个非子叶节点开始依次向上的非子叶节点进行遍历, 但是变成大堆之后, 交换首尾节点后他就又要进行变大堆了, 但是这次变大堆,找非子叶节点时候是从上节点往下找(这样些许会更快)

```java
public void heapSort(int [] arr)
{
    // 先首次将它变成大堆
    for(int i = arr.length / 2 - 1; i >= 0; i --)
    {
        // 非子叶节点先开始从下至上
        adjust(arr, i, arr.length);
    }
   // 然后在进行首尾交换
    for(int j = arr.length - 1; j > 0; j --)
    {
        int temp = arr[0];
        arr[0] = arr[j];
        arr[j] = temp;
        adjust(arr, 0, j);
    }
}
// i 代表的是要查询的非子叶节点
// length 代表需要变成大堆二叉树的长度 因为 长度 后来会变得越来越小
private void adjust(int []arr, int i, int length)
{
    int temp = arr[i]; // 暂时存储 arr[i] 因为非子叶节点要用来作比较会变换位置 ,到最后再用temp 把换到对应的位置
    for(int k = i * 2 + 1; k < length; k = k * 2 + 1)
    {
        if(k + 1 < length && arr[k] < arr[k + 1])
         k ++;   // 如果右大于左指针向右移
        if(temp < arr[k])
        {
            arr[i] = arr[k];
            i = k;  // i 指向 k 继续以 k 为非子叶节点进行遍历
        }else break;  // 因为是从下往上遍历 所以如果temp > arr[k] 代表已经变成大堆了 没必要继续了
    }
    arr[i] = temp; // 最后在将 i 换回来
}		

```
